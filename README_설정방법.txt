
✅ [시공보고서 웹앱 설정 방법]

1. Google 스프레드시트를 새로 만듭니다.
2. '확장 프로그램' > 'Apps Script' 선택 후 code.gs 내용 붙여넣기
3. code.gs 내 YOUR_SPREADSHEET_ID 부분에 시트 ID 입력
4. Apps Script에서 '배포' > '웹 앱으로 배포' → 누구나 접근으로 설정 후 URL 발급
5. script.js 파일 내 YOUR_SCRIPT_WEBAPP_URL 부분을 복사한 URL로 교체
6. index.html 파일을 웹 브라우저에서 열어 사용

※ 엑셀 시트 연동을 위해 Google 계정 권한 부여 필요
